    📖 Getting Started
    ✅
    Easiest Way: Double-click START_SERVER.bat in the project folder

    Or start manually:

    1

    Open Command Prompt

    Press Windows + R, type cmd, press Enter

    2

    Navigate to project

    cd c:\Users\srakella\ops-assistant
    3

    Start server

    .venv\Scripts\python.exe -m uvicorn main:app --port 8000
    4

    Wait for startup

    Wait until you see: "Application startup complete"

    ✍️ How to Fill the Form
    Field What to Write Example
    Title Short description (5-10 words) "Website is very slow"
    Description When, who affected, what happens "Since 2 PM, 50 users can't open the site"
    Errors Any error messages (optional) "Error 500" or leave empty
    📊 Understanding Priority Levels
    Priority Meaning Action Required
    P1 Critical - Major outage Fix immediately!
    P2 High - Significant impact Fix within hours
    P3 Medium - Limited impact Fix within 1-2 days
    P4 Low - Minimal impact Fix when possible
    🔧 Troubleshooting
    🔴
    "Server not connected"
    Server isn't running. Double-click START_SERVER.bat
    ⏳
    First request is slow (30+ sec)
    Normal! AI model is loading. Subsequent requests are faster.
    🔌
    "Address already in use"
    Server already running, or use different port: --port 8001
    ⚙️

    API Reference

    📍
    Base URL: http://localhost:8000
    Interactive Docs: http://localhost:8000/docs
    Method Endpoint Description
    GET /health Check server status
    POST /api/analyze Full analysis (triage + RCA + resolution)
    POST /api/triage Priority classification only
    POST /api/rca Root cause analysis only
    POST /api/resolution Resolution recommendations only
    POST /api/chat Free-form chat with AI
    GET /api/knowledge/stats Knowledge base statistics
    POST /api/ingest Add document to knowledge base

    Example Request

    curl -X POST http://localhost:8000/api/analyze \ -H "Content-Type: application/json" \ -d '{ "ticket_id": "INC001", "title": "Website is slow", "description": "Users reporting slow page loads since 2PM", "logs": "ERROR: Connection timeout" }'
